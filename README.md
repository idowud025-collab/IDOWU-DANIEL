# Library Book Management System

## Project Description
This is a simple Python-based Library Book Management System developed for SEN 201 assignment.
The program allows a user to enter book details and check the availability status.

## Features
- Add book title
- Add author name
- Check availability status
- Display book information

## Technologies Used
- Python

## How to Run the Program
1. Install Python on your system
2. Download or clone the repository
3. Run the program using:
   python library_book_management_system.py
